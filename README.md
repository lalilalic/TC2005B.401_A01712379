# Laboratorio 8 - Node.js

## Descripcion

Este laboratorio contiene ejemplos basicos de Node.js y una pequena aplicacion web.

## Contenido

- `app.js`: ejercicios solicitados para ejecutarse en consola con Node.
- `server.js`: servidor web basico hecho con Node.
- `old_labs/index.html`: pagina de un laboratorio anterior que el servidor entrega al cliente.

## Instrucciones de ejecucion

### 1. Ejecutar ejercicios en consola

```bash
node app.js
```

Se muestran en consola:

- el promedio de un arreglo de numeros
- la escritura de un string en `hola.txt`
- la solucion de un problema extra en JavaScript sobre Node

### 2. Ejecutar servidor web

```bash
node server.js
```

Despues abre en el navegador:

```text
http://localhost:3000
```

## Evidencia esperada

- En consola debe aparecer el resultado de los ejercicios.
- Debe generarse o actualizarse el archivo `hola.txt`.
- El servidor debe responder con la pagina ubicada en `old_labs/index.html`.
