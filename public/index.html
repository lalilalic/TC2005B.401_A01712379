Pregunta

¿Qué ventajas y desventajas tiene la integración de tus aplicaciones web con servicios web desarrollados por terceros?

Una ventaja es que no tenemos que programar todo desde cero y podemos usar funciones que ya existen.

Una desventaja es que dependemos de otro servicio. Si el API cambia o deja de funcionar, nuestra aplicación también puede tener problemas.
